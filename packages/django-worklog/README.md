# django-worklog

Server-side work session timing for operational work tracking.

## Install

```bash
pip install django-worklog
```

## Quick Start

1. Add `django_worklog` to `INSTALLED_APPS`
2. Run migrations: `python manage.py migrate django_worklog`
3. Use WorkSession to track timed work items
