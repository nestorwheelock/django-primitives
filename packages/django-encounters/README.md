# django-encounters

Domain-agnostic encounter state machine with pluggable validators.

## Install

```bash
pip install django-encounters
```

## Quick Start

1. Add `django_encounters` to `INSTALLED_APPS`
2. Run migrations: `python manage.py migrate django_encounters`
3. Import and use the encounter models

## Versioning

This package follows semantic versioning. See `__version__` in the package root.
